De CSS regel heeft geen effect wanneer deze toegepast wordt op een <td> cel in een <tr> rij met de 'alt' class, omdat de 'alt' class veel specifieker gericht is dan de 'onbekend' class. Als je de 'onbekend' class specifieker maakt dan de 'alt' class, dan zal de 'onbekend' class effect hebben op de cel. Je kunt ook de class even specifiek maken en ervoor zorgen dat de 'onbekend' class verder in de css file staat dan de 'alt' class.
