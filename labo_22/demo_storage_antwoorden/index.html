1. Globale variabelen:
    • De data blijft niet behouden als je de pagina herlaadt
    • De data blijft niet behouden als je het tabblad sluit en de pagina opnieuw opent
    • Twee tabbladen die dezelfde pagina tonen gebruiken niet dezelfde data
  2. Session storage:
    • De data blijft niet behouden als je het tabblad sluit en de pagina opnieuw opent
    • De data blijft behouden als je de pagina herlaadt
    • de data wordt niet gedeeld door verschillende tabbladen als je de pagina meermaals opent
    • Ik denk dat een andere file of url aan de data zou kunnen geraken als je ze in hetzelfde tabblad opent
  3. Local storage:
    • De data blijft behouden als je het tabblad sluit en de pagina opnieuw opent
    • De data blijft behouden als je de pagina herlaadt
    • De data blijft behouden als je de browser afsluit, opnieuw start en de pagina weer opent
    • De data blijft niet behouden als je de pagina in een andere browser opent
    • De data wordt gedeeld door verschillende tabbladen als je de pagina meermaals opent
